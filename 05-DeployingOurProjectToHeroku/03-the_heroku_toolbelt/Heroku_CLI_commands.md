### Install Heroku within Gitpod:
- `npm install -g Heroku`


### Log In to Heroku:
- `heroku login -i`


### View your Heroku Apps:
- `heroku apps`


### Rename a Heroku App:
- `heroku apps:rename NEW-NAME --app CURRENT-APP-NAME`


### Deployed Heroku App URL:
- `https://YOUR-APP-NAME.herokuapp.com`
